# 🧠 Personality Detection & Career Suggestion System

## 📌 Project Overview
This project predicts personality (Introvert/Extrovert) from user text using NLP and suggests suitable careers.

## 🚀 Features
- Personality prediction using Machine Learning
- Career recommendations
- Confidence score display
- Interactive UI using Streamlit

## 🛠️ Technologies Used
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
